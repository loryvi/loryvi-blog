# Loryvi's personal website (https://lory-blog.vercel.com)

Welcome to my personal website and portfolio. 🛸

- 💾 **Hosted** by [Vercel](https://vercel.com)
- 📝 **Writing** articles with [MDX](https://mdxjs.com/)
